# 🧠 AI Content Text Moderation System

An intelligent **AI-powered text moderation system** designed to automatically detect, classify, and filter inappropriate or harmful content across multiple categories using **Natural Language Processing (NLP)** and **Machine Learning**.

---

## 🚀 Features

- 🔍 **Multi-category Detection**
  - Toxicity
  - Hate Speech
  - Offensive Language
  - Spam / Abuse

- 🌍 **Multilingual Support**
  - Language detection
  - Automatic translation for consistent moderation

- 🤖 **Transformer-based Models**
  - Uses pre-trained models from Hugging Face

- 📊 **Sentiment Analysis**
  - Analyze emotional tone of the text

- ⚙️ **Real-time Moderation**
  - Can be integrated into live applications

- 📈 **Performance Metrics**
  - Accuracy
  - Precision
  - Recall
  - F1-score

---

## 🛠️ Tech Stack

- Python  
- Transformers (Hugging Face)  
- NLTK  
- TextBlob  
- LangDetect  
- Deep Translator  
- Scikit-learn  

---

## 📂 Project Workflow

1. **Input Text**  
2. **Language Detection & Translation**  
3. **Text Preprocessing**  
4. **Model Prediction**  
5. **Output Classification & Moderation Decision**
